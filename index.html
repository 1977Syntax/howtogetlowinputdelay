1. Use an Ethernet Cable

Connecting your PC or console directly to the router using an Ethernet cable is crucial for reducing ping and input delay. This connection type provides a more stable and faster internet connection compared to Wi-Fi.

2. PC Cleanup

Regularly clean your PC of unnecessary files, viruses, and malware using software like Malwarebytes and Advanced SystemCare. This will help enhance performance and reduce latency.

3. Change DNS Settings

Switching your DNS to Google’s DNS can potentially improve latency. Use the following settings:

  • Preferred DNS: 8.8.8.8
  • Alternate DNS: 8.8.4.4

4. Use TCP Optimizer

TCP Optimizer is a free tool that helps optimize your internet connection settings. Run it as an administrator and follow the recommended settings for the best results.

5. Registry Settings

Adjusting your registry settings can improve performance. Use a provided registry file and merge it to apply better configurations.
